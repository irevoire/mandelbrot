# Parallélisation du calcul d'ensembles de Mandelebrot

L'objet du TP est de prosuire des versions parallèles du calcul de l'ensemble de Mandelbrot.

Une version séquentielle est fournie dans le fichier `mandel-seq.c`.

Deux objectifs progressifs à ce TP :
1. produire une version parallèle
2. se soucier de l'équilibre de la charge
